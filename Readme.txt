# Thrive Remote

Thrive Remote is a career and talent platform that connects professionals with remote job opportunities.  
The platform provides a streamlined way for candidates to apply for roles and for the team to receive and manage applications.

## Features
- 🌐 **Company Website** – Live at [www.thrive-remote.com](https://www.thrive-remote.com)
- 👥 **Careers Page** – Showcases open roles with descriptions
- 📄 **Job Application Form** – Applicants can apply directly and upload resumes
- ✉️ **Contact Form** – For general inquiries and communication
- 📧 **Email Notifications** – Applications and contact messages are forwarded to the admin inbox
- 📂 **Resume Uploads** – Stored securely in `/uploads` for reference

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (Bootstrap template)
- **Backend:** Node.js, Express.js
- **Email Handling:** Nodemailer (SMTP)
- **File Uploads:** Multer
